# 人皆有不忍人之心

## 原文与译文

> **〔第1段〕 孟子曰："人皆有不忍人之心。** 先王有**不忍人**之心，**斯**有**不忍人**之政矣；以**不忍人**之心，行**不忍人**之政，治天下可**运之掌上**。

📝 孟子说："每个人都有怜恤别人的心。先王因为有怜恤别人的心，于是就有怜恤别人的政治了；用怜恤别人的心，施行怜恤别人的政治，治理天下就可以像在手掌上运转一样容易。

---

> **〔第2段〕** 所以谓人皆有**不忍人**之心者：**今人乍见孺子将入于井，皆有怵惕恻隐之心**——非所以**内交[[03-知识点/虚词/于|**于**]]孺子**之父母也，非所以**要誉[[03-知识点/虚词/于|**于**]]乡党**朋友也，非[[03-知识点/实词/恶|**恶**]]其声而**然**也。由是观之，无**恻隐**之心，非人也；无羞[[03-知识点/实词/恶|**恶**]]之心，非人也；无辞让之心，非人也；无是非之心，非人也。**恻隐之心，仁之端也；羞恶之心，义之端也；辞让之心，礼之端也；是非之心，智之端也。** 人之有是四**端**也，犹其有**四体**也。有是四**端**而自谓不能者，自[[03-知识点/实词/贼|**贼**]]者也；谓其君不能者，[[03-知识点/实词/贼|**贼**]]其君者也。

📝 之所以说每个人都有怜恤别人的心，道理在于：**假如现在有人忽然看到一个孩子快要掉进井里，都会有惊恐怜悯的心**——这并不是想借此和孩子的父母结交，并不是想借此在乡邻朋友中博取声誉，也不是因为讨厌孩子的哭叫声才这样的。由此看来，没有怜悯之心，不算人；没有羞耻憎恶之心，不算人；没有谦辞礼让之心，不算人；没有辨别是非之心，不算人。**怜悯之心，是仁的发端；羞耻憎恶之心，是义的发端；谦辞礼让之心，是礼的发端；辨别是非之心，是智的发端。** 人有这四个发端，就像人有四肢一样。有这四端却说自己不行的人，是自己戕害自己；说他的君主不行的人，是戕害他的君主。

---

> **〔第3段〕** 凡有四**端**于我者，知皆**扩而充之**矣，若火之始**然**，泉之始**达**。**苟能充之，足以保四海；苟不充之，不足以事父母。**"

📝 凡是自身具有这四端的人，如果知道把它们都扩充起来，就会像火刚燃烧起来、泉水刚涌流出来一样。**如果能够扩充它们，就足以安定天下；如果不能扩充它们，就连父母都侍奉不了。**"

## 文章结构与赏析

### 结构梳理

```
提出论点（第1段）    → "人皆有不忍人之心"（性善论的基石）

举例论证（第2段）    → 孺子入井——人人本能地生发恻隐之心
                        （非功利、非名誉、非感官——纯粹道德本能）
                          ↓
                     → "四心"推导：
                       恻隐之心 → 仁
                       羞恶之心 → 义
                       辞让之心 → 礼
                       是非之心 → 智
                          ↓
                     → "四端"说：四心是仁义礼智的萌芽

引申论证（第3段）    → "扩而充之"——四端需要培育和发扬
                       苟能充之 → 保四海
                       苟不充之 → 不足以事父母
```

### 主旨

通过"孺子入井"的经典案例，论证"**人皆有不忍人之心**"，由此提出"**四端**"说——恻隐、羞恶、辞让、是非之心分别是仁、义、礼、智的萌芽，并强调"**扩而充之**"的重要性。这是孟子"性善论"的核心论述。

### 艺术特色

1. **举例生动**："孺子入井"——设身处地，人人可以验证，论证力极强
2. **排比递进**："无恻隐之心非人也……"→"恻隐之心仁之端也……"——句式整齐而有气势
3. **比喻精妙**：四端如"火之始然，泉之始达"——形象说明善端需要培育扩充
4. **逻辑严谨**：由性善推出仁政——以不忍人之心行不忍人之政

## 文化常识

- **孟子**（约前372—前289），名轲，字子舆，战国时期儒家代表人物，与孔子并称"孔孟"
- **"性善论"**：孟子学说的核心，主张人性本善，"四端"人人皆有
- **"仁政"**：孟子将性善论引申到政治领域，主张君王应以"不忍人之心"行"不忍人之政"
- **"四端"**：恻隐之心（仁）、羞恶之心（义）、辞让之心（礼）、是非之心（智）
- **"扩而充之"**：孟子认为善端需要后天的培育和扩充，否则会泯灭
- **乡党**：古代以一万二千五百家为乡，五百家为党，泛指乡里
