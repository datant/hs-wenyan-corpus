# 游园·【皂罗袍】

## 原文与译文

> **〔第1段〕** **原来姹紫嫣红开遍，似这般都付与断井颓垣。**
> **良辰美景奈何天，赏心乐事谁家院！**

📝 原来这里百花盛开、万紫千红，却都托付给了破败的枯井和倒塌的院墙。这样美好的时光、美丽的景色，可苍天对我如此不公；那赏心悦目、快乐的事，又落在谁家的庭院呢！

---

> **〔第2段〕** **朝飞暮卷，云霞翠轩；雨丝风片，烟波画船——**
> **锦屏人忒看的这韶光贱！**

📝 清晨的飞云、黄昏的卷帘，彩霞映照着华丽的亭台；细雨如丝、微风片片，烟雾笼罩着彩绘的画船——只是深闺中人太把这美好春光看得轻贱了！

## 注释

| 词语 | 释义 |
|------|------|
| 姹紫嫣红 | 各种花朵色彩娇艳 |

| 良辰美景 | 美好的时光和景色 |
| 奈何天 | 对苍天无可奈何 |
| 赏心乐事 | 赏心悦目、快乐的事 |
| 朝飞暮卷 | 化用王勃"画栋朝飞南浦云，珠帘暮卷西山雨" |

## 赏析

**出处**：选自汤显祖《牡丹亭·惊梦》，是女主人公杜丽娘游园时的一段唱词，通过游园感叹春光易逝、青春被禁锢。

**情感层次**：
1. **"原来姹紫嫣红开遍"**——见到满园春色的惊喜
2. **"似这般都付与断井颓垣"**——转悲，美好的春光无人欣赏
3. **"良辰美景奈何天，赏心乐事谁家院"**——感叹自己的不幸命运，青春被禁锢在深闺
4. **"锦屏人忒看的这韶光贱"**——深闺人把春光看贱了，实则是对封建礼教的委婉抗议

**艺术特色**：
- **以乐景写哀情**：满园春色反衬杜丽娘的孤独与无奈
- **用典自然**："朝飞暮卷"化用王勃《滕王阁序》名句
- **语言优美**：色彩鲜明（姹紫嫣红、云霞、烟波），意境深远
- **情景交融**：春光之美 ↔ 青春被压抑之悲

**思想意义**：杜丽娘的感叹反映了明代女性追求个性解放、冲破封建礼教束缚的觉醒意识。

**名句**：
> 原来姹紫嫣红开遍，似这般都付与断井颓垣。
>
> 良辰美景奈何天，赏心乐事谁家院！
