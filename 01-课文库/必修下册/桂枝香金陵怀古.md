# 桂枝香·金陵怀古

## 原文与译文

> **〔上阕〕** **登临送目，正故国晚秋，天气初肃。**
> **千里澄江似练，翠峰如簇。**
> **归帆去棹残阳里，背西风，酒旗斜矗。**
> **彩舟云淡，星河鹭起，画图难足。**

📝 登上高处极目远望，正是故都金陵深秋时节，天气开始萧瑟。千里长江澄澈如白练，青翠的山峰像箭头般耸立。夕阳中归来的船只在残阳里穿行，西风斜挂酒旗。彩色船只笼罩着薄云淡雾，白鹭在银河般的长江上起飞，这样的景色画也难以完美描画。

---

> **〔下阕〕** **念往昔，繁华竞逐，叹门外楼头，悲恨相续。**
> **千古凭高，对此谩嗟荣辱。**
> **六朝旧事随流水，但寒烟衰草凝绿。**
> **至今商女，时时犹唱，后庭遗曲。**

📝 想起往昔，这里的人们争相追逐繁华，可叹门外敌军兵临城下，楼头君王还在享乐，悲恨接连不断。千百年来登高凭吊的人们，对此空叹兴衰荣辱。六朝旧事已随流水逝去，只有寒烟衰草还凝聚着绿色。至今歌女们还时时吟唱那《后庭》遗曲。

## 注释

| 词语 | 释义 |
|------|------|
| 登临送目 | 登高远望 |
| 故国 | 故都（指金陵） |
| 肃 | 萧瑟、肃杀 |
| 澄江似练 | 清澈的长江像白绸带 |
| 簇（cù） | 箭头（指山峰如箭） |
| 归帆去棹 | 过往的船只 |
| 酒旗斜矗 | 酒店外的旗帜斜挂 |
| 星河鹭起 | 长江上的白鹭起飞（星河喻长江） |
| 门外楼头 | 化用杜牧"门外韩擒虎，楼头张丽华" |
| 谩嗟 | 空叹 |
| 后庭遗曲 | 即《玉树后庭花》，陈后主所作，世称亡国之音 |

## 赏析

**上片写景**：
- "登临送目"总领全篇
- "千里澄江似练，翠峰如簇"——远景，江如白练、山如箭簇
- "归帆去棹残阳里"——中景，日落归帆
- "彩舟云淡，星河鹭起"——近景，化静为动
- "画图难足"——感叹美景难以描画

**下片怀古**：
- "念往昔"由景入史
- "门外楼头"——用陈后主与张丽华典故，讽刺亡国之君
- "六朝旧事随流水"——物是人非，历史无情
- "至今商女，时时犹唱，后庭遗曲"——化用杜牧《泊秦淮》"商女不知亡国恨"，警示当世

**主旨**：借六朝兴亡的历史教训，警示北宋统治者不要重蹈覆辙，有强烈的现实关怀。
