# 念奴娇·过洞庭

## 原文与译文

> **〔上阕〕** **洞庭青草，近中秋、更无一点风色。**
> **玉鉴琼田三万顷，着我扁舟一叶。**
> **素月分辉，明河共影，表里俱澄澈。**
> **悠然心会，妙处难与君说。**

📝 洞庭湖和青草湖，临近中秋时节，湖面上没有一丝风。如玉的月光洒在三万顷湖面上，我的小船飘荡其间。明月分洒着光辉，银河与月影同在，天上和湖里都是一片澄明清澈。悠然心领神会，这美妙的境界难以用语言表达。

---

> **〔下阕〕** **应念岭表经年，孤光自照，肝肺皆冰雪。**
> **短发萧骚襟袖冷，稳泛沧浪空阔。**
> **尽挹西江，细斟北斗，万象为宾客。**
> **扣舷独啸，不知今夕何夕！**

📝 回想在岭南任职的一年，只有明月清辉照着我，我的肝胆都像冰雪一样晶莹。头发稀疏、衣衫单薄，安稳地泛舟在广阔的碧波之上。舀尽西江之水，斟满北斗做酒杯，邀请天地万物来做客。敲着船舷独自长啸，不知今晚是什么夜晚！

## 注释

| 词语 | 释义 |
|------|------|
| 青草 | 青草湖，与洞庭湖相连 |
| 风色 | 风 |
| 玉鉴琼田 | 如玉的镜子、如玉的田地（喻月色下的湖面） |
| 著（zhuó） | 附着、安放 |
| 素月 | 洁白的月亮 |
| 明河 | 银河 |
| 表里俱澄澈 | 湖面与湖心都清澈透明（亦喻内心纯洁） |
| 岭表 | 五岭以南（指岭南地区） |
| 经年 | 一年 |
| 孤光 | 月光 |
| 肝肺皆冰雪 | 肝胆如冰雪般晶莹（喻高洁品格） |
| 萧骚 | 稀疏 |
| 沧浪 | 青苍色的水 |
| 尽挹（yì）西江 | 舀尽西江之水（比喻气魄宏大） |
| 细斟北斗 | 把北斗星当作酒器细细斟酒 |
| 万象为宾客 | 天地万物都是客人 |
| 扣舷 | 敲击船舷 |

## 赏析

**上片写景**：
- "洞庭青草，近中秋、更无一点风色"——开篇宁静
- "玉鉴琼田三万顷"——极写湖面之广阔澄澈
- "素月分辉，明河共影，表里俱澄澈"——天人合一，月光银河与湖面交相辉映
- "悠然心会，妙处难与君说"——不可言传的美妙体验

**下片抒情**：
- "肝肺皆冰雪"——表明自己品行高洁、光明磊落
- "尽挹西江，细斟北斗，万象为宾客"——气魄宏大，奇思妙想，邀天地万物共饮
- "扣舷独啸，不知今夕何夕"——物我两忘，超越尘世

**创作背景**：张孝祥因谗言被罢官，从桂林北归途经洞庭湖，面对浩瀚湖景，写下此词，虽被贬谪却无颓丧之气，反而气度超然。

**名句**：
> 素月分辉，明河共影，表里俱澄澈。
>
> 尽挹西江，细斟北斗，万象为宾客。
>
> 扣舷独啸，不知今夕何夕。
