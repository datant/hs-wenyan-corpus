# 谏太宗十思疏

## 原文与译文

> **〔第1段〕** 臣闻求木之长者，必固其根本；欲流之远者，必**浚**其泉源；思国之安者，必积其德义。源不深而望流之远，根不固而求木之长，德不厚而思国之理，臣虽下愚，知其不可，而况于明哲乎！人君**当神器之重**，**居域中之大**，将崇极天之峻，永保无疆之**休**。不念居安思危，**戒奢以俭**，德不处其厚，情不胜其欲，斯亦伐根以求木茂，塞源而欲流长者也。

📝 我听说想要树木长得好，一定要使它的根牢固；想要水流得远，一定要疏通它的源头；想要国家安定，一定要积累德义。源头不深却希望水流得远，根不牢固却想要树木长得好，德行不深厚却想要国家治理好，我虽然愚笨，也知道这是不可能的，何况明智的人呢！国君担负着帝位的重任，处于天地间最高的地位，想要推崇皇权的高峰，永保无穷的福禄。如果不考虑居安思危，用节俭来戒除奢侈，德行不能保持深厚，性情不能克服欲望，这也就像砍断树根来求树木茂盛，堵塞源头来想要水流长远一样。

---

> **〔第2段〕** 凡百元首，承天**景命**，莫不**殷忧**而道著，功成而德衰。有善始者实繁，能**克终**者盖寡。岂其取之易而守之难乎？昔取之而有余，今守之而不足，何也？夫在**殷忧**必竭诚以待下，既得志则纵情以**傲物**。竭诚则胡越为一体，**傲物**则骨肉为行路。虽**董**之以严刑，**振**之以威怒，终苟免而不怀仁，貌恭而不心服。怨不在大，可畏惟人；**载舟覆舟**，所宜深慎；**奔车朽索**，其可忽乎！

📝 历代所有帝王，承受上天的大命，没有不在深忧时德行显著，但功业建成后德行就衰微了。开头做得好的确实很多，能够坚持到底的大概很少。难道是夺取天下容易而守住天下困难吗？过去夺取天下时力量有余，现在守住天下却力量不足，为什么呢？因为深忧时一定竭尽诚意来对待下属，得志后就放纵情欲来傲视他人。竭尽诚意，即使像胡越那样敌对的人也能结为一体；傲视他人，即使是骨肉至亲也会成为陌路人。即使用严酷的刑罚来督责，用威严的怒斥来震慑，最终也不过是使人苟且免于罪责而并不心怀仁德，表面恭顺而内心不服。怨恨不在大小，可怕的是民众（的怨气）。水能载舟也能覆舟，这是应该深切谨慎的。用腐烂的绳索驾驭飞奔的马车，难道可以疏忽吗？

---

> **〔第3段〕** 君人者，诚能**见可欲**则思知足以自戒，将有作则思**知止**以安人，念高危则思**谦冲[[03-知识点/虚词/而|**而**]]自牧**，惧满溢则思**江海下百川**，乐**盘游**则思**三驱**以为度，忧懈怠则思慎始而敬终，虑**壅蔽**则思虚心以纳下，想谗邪则思正身以**黜恶**，恩所加则思无因喜以**谬赏**，罚所及则思无因怒而**滥刑**。总此十思，弘兹九德，**简能**而任之，择善而从之，则智者尽其谋，勇者竭其力，仁者播其惠，信者效其忠。文武争驰，君臣无事，可以尽豫游之乐，可以养松乔之寿，鸣琴**垂拱**，不言而化。何必劳神苦思，**代下司职**，役聪明之耳目，亏无为之大道哉！

📝 做国君的人，如果见到能引起自己喜好的东西，就想到要知足来警戒自己；将要兴建土木，就想到要适可而止来安定百姓；想到地位高而险，就想到要谦虚平和来加强自我修养；担心自满，就想到要像江海一样甘居百川之下；喜好游乐，就想到要以三面驱鸟作为限度；担忧松懈，就想到要谨慎地开始、敬慎地结束；担心耳目被堵塞蒙蔽，就想到要虚心接纳臣下的意见；考虑到有谗佞奸邪之人，就想到要端正自身来斥退奸恶；施加恩惠时，就想到不要因为自己高兴而错误地赏赐；施行处罚时，就想到不要因为自己发怒而滥用刑罚。总共这"十思"，弘扬这"九德"，选拔有才能的人而任用他们，选择好的意见而听从它们，那么聪明的人就能全部献出他们的谋略，勇敢的人就能用尽他们的力量，仁爱的人就能传播他们的恩惠，诚信的人就能献出他们的忠诚。文臣武将争相效力，君臣之间相安无事，就可以尽情享受游乐的快乐，可以颐养得像松柏一样长寿，弹琴垂衣，不用说话而天下教化。何必劳神苦思，代替下属管理职事，役使自己的耳目，损害无为而治的大道呢！

## 文章赏析

**十思内容**：

| 十思 | 应对的问题 | 修身方面 |
|------|-----------|---------|
| 见可欲——知足以自戒 | 贪欲 | 戒贪 |
| 将有作——知止以安人 | 大兴土木 | 戒奢 |
| 念高危——谦冲自牧 | 骄傲 | 戒骄 |
| 惧满溢——江海下百川 | 自满 | 戒满 |
| 乐盘游——三驱为度 | 游乐过度 | 戒逸 |
| 忧懈怠——慎始敬终 | 松懈 | 戒怠 |
| 虑壅蔽——虚心纳下 | 听不到真话 | 戒蔽 |
| 想谗邪——正身黜恶 | 小人谗言 | 戒谗 |
| 恩所加——无因喜谬赏 | 赏赐不当 | 戒私 |
| 罚所及——无因怒滥刑 | 刑罚不当 | 戒怒 |

**论证特色**：
1. **比喻开篇**：求木之长必固根本、欲流之远必浚泉源——引出主题
2. **对比论证**：善始者繁 vs 克终者寡；取之有余 vs 守之不足
3. **排比有力**："十思"排比句式，气势充沛
4. **历史经验**："载舟覆舟"——以民为本的治国理念
