# 登岳阳楼

## 原文与译文

> **〔上阕〕** **昔闻洞庭水，今上岳阳楼。**
> **吴楚东南坼，乾坤日夜浮。**

📝 从前就听说过洞庭湖的盛名，今天终于登上了岳阳楼。洞庭湖将吴楚两地分割在东南，天地万物仿佛日夜漂浮在湖面。

---

> **〔下阕〕** **亲朋无一字，老病有孤舟。**
> **戎马关山北，凭轩涕泗流。**

📝 亲朋好友没有一点音信，年老多病只有一只孤舟相伴。关山以北战火未息，倚着栏杆我泪流满面。

## 注释

| 词语 | 释义 |
|------|------|
| 洞庭水 | 洞庭湖 |
| 吴楚 | 春秋战国时两国名，吴地在东，楚地在西 |
| 坼（chè） | 分裂、分开 |
| 乾坤 | 天地、日月 |
| 字 | 书信、音信 |
| 戎马 | 战马，借指战争 |
| 凭轩 | 倚靠栏杆 |
| 涕泗 | 眼泪和鼻涕 |

## 赏析

**意境**：此诗意境阔大而情感深沉，是杜甫五言律诗的压卷之作。

**首联**："昔闻"与"今上"相对，既有夙愿得偿的欣慰，也暗含世事沧桑之感。

**颔联（名句）**："吴楚东南坼，乾坤日夜浮"——极写洞庭湖的浩瀚壮阔，"坼""浮"二字力敌千钧，将湖水写得仿佛分裂大地、浮载天地。

**颈联**：从壮阔之景转入个人身世——"亲朋无一字，老病有孤舟"，漂泊孤独。

**尾联**："戎马关山北"——由个人际遇转向国家危难，思乡忧国，"凭轩涕泗流"收束全篇。

**艺术特色**：全诗由登楼写景起笔，境界由大到小（天地→孤舟→自身），再由小到大（自身→国家），情景交融，境界博大。
