# 判断句

> 判断句是对事物的性质、情况、事物之间的关系做出肯定或否定判断的句子。现代汉语常用"是"表判断，文言文中则用多种结构。

---

## 一、基本类型

### 1. "……者，……也" 式（最常见）

| 例句 | 出处 |
|------|------|
| 屈原者，名平，楚之同姓也 | [[屈原列传]] |
| 缑王者，昆邪王姊子也 | [[苏武传]] |
| 师者，所以传道受业解惑也 | [[师说]] |

### 2. "……，……也" 式

| 例句 | 出处 |
|------|------|
| 项脊轩，旧南阁子也 | [[项脊轩志]] |
| 梁，吾仇也 | [[五代史伶官传序]] |
| 妪，先大母婢也 | [[项脊轩志]] |
| 此世所以不传也 | [[石钟山记]] |
| 是造物者之无尽藏也 | [[赤壁赋]] |

### 3. "……者，……" 式

| 例句 | 出处 |
|------|------|
| 天下者，高祖天下 | — |

### 4. "……者也" 式

| 例句 | 出处 |
|------|------|
| 沛公之参乘樊哙者也 | [[鸿门宴]] |

### 5. 用"乃""即""则""皆""非"等副词表判断

| 例句 | 标志词 | 出处 |
|------|--------|------|
| 是**乃**仁术也 | 乃 | [[齐桓晋文之事]] |
| **即**谋单于，何以复加 | 即 | [[苏武传]] |
| 此**则**岳阳楼之大观也 | 则 | — |
| 所处**皆**是也 | 皆 | [[石钟山记]] |
| 人**非**生而知之者 | 非（否定） | [[师说]] |
| **非**蛇鳝之穴无可寄托者 | 非（否定） | [[劝学]] |
| 无恻隐之心，**非**人也 | 非（否定） | [[人皆有不忍人之心]] |

### 6. "为" 字判断

| 例句 | 出处 |
|------|------|
| 如今人方为刀俎，我为鱼肉 | [[鸿门宴]] |
| 吾令人望其气，皆为龙虎 | [[鸿门宴]] |

### 7. 无标志判断句（直接判断）

| 例句 | 出处 |
|------|------|
| 臣密今年四十有四 | [[陈情表]] |
| 刘备天下枭雄 | —（《赤壁之战》） |
| 今臣亡国贱俘 | [[陈情表]] |

## 二、"是"字在文言中的特殊用法

文言文中的"是"多数情况下是**指示代词**（"这"），不是判断词：

| 例句 | 释义 |
|------|------|
| **是**说也，人常疑之 | 这（种说法） |
| **是**夕始觉有迁谪意 | 这（一晚） |
| **是**日也，天朗气清 | 这（一天） |

偶尔"是"也作判断词（汉代以后渐多）：
- 问今**是**何世（《桃花源记》）
- 自言本**是**京城女（[[琵琶行]]）

---

## 三、翻译方法

文言判断句在翻译时通常要加"是"：

- 原文：师者，所以传道受业解惑也
- 译文：老师，**是**用来传授道理、教授学业、解答疑惑的人
