# 倒装句

> 文言文中，为了强调或由于语言习惯，某些句子成分的位置与现代汉语不同，形成"倒装"。主要分为四种：**宾语前置、定语后置、状语后置、主谓倒装**。

---

## 一、宾语前置

宾语出现在动词或介词之前。

### 1. 否定句中代词作宾语前置

规则：否定词（不/未/莫/无）+ 代词宾语 + 动词

| 例句 | 前置结构 | 出处 |
|------|----------|------|
| 古之人**不余欺**也 | 不欺余 | [[石钟山记]] |
| **不我若**也 | 不若我 | [[种树郭橐驼传]] |
| 莫**之**能御也 | 莫能御之 | [[齐桓晋文之事]] |
| 自书典所记，未**之**有也 | 未有之 | — |

### 2. 疑问句中疑问代词作宾语前置

规则：疑问词（何/安/谁/奚/焉）+ 动词/介词

| 例句 | 前置结构 | 出处 |
|------|----------|------|
| **何**为其然也 | 为何 | [[赤壁赋]] |
| 而今**安**在哉 | 在安 | [[赤壁赋]] |
| 而又**何**羡乎 | 羡何 | [[赤壁赋]] |
| **何以**汝为见 | 以何 | [[苏武传]] |
| 大王来**何**操 | 操何 | [[鸿门宴]] |
| 沛公**安**在 | 在安 | [[鸿门宴]] |
| **奚**惆怅而独悲 | 为何惆怅 | [[归去来兮辞]] |
| 复驾言兮**焉**求 | 求焉 | [[归去来兮辞]] |
| 乐夫天命复**奚**疑 | 疑奚 | [[归去来兮辞]] |
| 当察乱**何自**起 | 自何 | [[兼爱]] |
| **恶**施不孝 | 施恶 | [[兼爱]] |
| 子卿尚复**谁为**乎 | 为谁 | [[苏武传]] |

### 3. 用"之""是"作标志的宾语前置

| 例句 | 标志 | 出处 |
|------|------|------|
| 句读**之**不知，惑**之**不解 | 之 | [[师说]] |
| 唯利**是**图 | 是 | — |
| 何陋**之**有 | 之 | — |

### 4. 介词宾语前置

| 例句 | 前置结构 | 出处 |
|------|----------|------|
| **何以**战 | 以何 | — |
| **是以**见放 | 以是（因此） | [[屈原列传]] |
| 余**是以**记之 | 以是 | [[石钟山记]] |
| **自**谓不能者 | 谓自 | [[人皆有不忍人之心]] |

## 二、定语后置

定语放在中心词之后，现代汉语定语在中心词之前。

### 1. "中心词+之+后置定语" 式

| 例句 | 正常语序 | 出处 |
|------|----------|------|
| 蚓无爪牙**之**利，筋骨**之**强 | 利之爪牙，强之筋骨 | [[劝学]] |
| 凌万顷**之**茫然 | 茫然之万顷 | [[赤壁赋]] |
| 仰观宇宙**之**大 | 大之宇宙 | [[兰亭集序]] |
| 俯察品类**之**盛 | 盛之品类 | [[兰亭集序]] |

### 2. "中心词+者" 式

| 例句 | 出处 |
|------|------|
| 客有吹洞箫**者** | [[赤壁赋]] |
| 宋人有善为不龟手之药**者** | [[五石之瓠]] |
| 石之铿然有声**者** | [[石钟山记]] |
| 求人可使报秦**者** | — |

### 3. "中心词+之+后置定语+者" 式

| 例句 | 出处 |
|------|------|
| 马**之**千里**者** | —（《马说》） |
| 石**之**铿然有声**者** | [[石钟山记]] |

## 三、状语后置（介宾短语后置）

文言文中，介宾短语常放在动词之后，现代汉语放在动词之前。这是最常见的一种倒装。

| 例句 | 后置结构 | 正常语序 | 出处 |
|------|----------|----------|------|
| 青，取之**于蓝** | 于蓝取之 | 从蓝草中提取 | [[劝学]] |
| 月出**于东山之上** | 于东山之上出 | 从东山上升起 | [[赤壁赋]] |
| 尝学琵琶**于穆、曹二善才** | 于善才学 | 向善才学习 | [[琵琶行]] |
| 会**于会稽山阴之兰亭** | 于兰亭会 | 在兰亭聚会 | [[兰亭集序]] |
| 申之**以孝悌之义** | 以孝悌之义申之 | 用孝悌的道理教导 | [[齐桓晋文之事]] |
| 盛**以锦囊** | 以锦囊盛 | 用锦囊装 | [[五代史伶官传序]] |
| 非尊**于**齐楚……之君也 | 非于齐君尊 | 不比齐君尊贵 | [[过秦论]] |
| 急于星火 | 于星火急 | 比流星还急 | [[陈情表]] |
| 农人告余**以春及** | 以春及告余 | 把春天到了告诉我 | [[归去来兮辞]] |
| 得双石**于潭上** | 于潭上得 | 在潭上找到 | [[石钟山记]] |
| 大声发**于水上** | 于水上发 | 从水上发出 | [[石钟山记]] |

---

## 四、主谓倒装

为强调谓语，把谓语放在主语之前。

| 例句 | 出处 |
|------|------|
| **甚矣**，汝之不惠 | —（《愚公移山》） |
| **贤哉**，回也 | —（《论语》） |

---

## 翻译方法

- **宾语前置** → 把宾语放回动词后：不余欺 → 不欺余
- **定语后置** → 把定语放到名词前：爪牙之利 → 锋利的爪牙
- **状语后置** → 把状语移到动词前：取之于蓝 → 从蓝草中提取
