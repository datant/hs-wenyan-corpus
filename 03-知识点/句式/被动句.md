# 被动句

> 被动句是主语是动作的承受者而不是发出者的句子。文言文被动句有明确的标志，与现代汉语"被"字句不完全相同。

---

## 一、有标志被动句

### 1. "于" 字式（介词"于"引进行为的主动者）

| 例句 | 主动者 | 出处 |
|------|--------|------|
| 内惑**于**郑袖，外欺**于**张仪 | 郑袖/张仪 | [[屈原列传]] |
| 此非孟德之困**于**周郎者乎 | 周郎 | [[赤壁赋]] |
| 不拘**于**时 | 时俗 | [[师说]] |
| 而智勇多困**于**所溺 | 所溺 | [[五代史伶官传序]] |

### 2. "见" 字式（"见"表被动）

| 例句 | 出处 |
|------|------|
| 信而**见**疑，忠而被谤 | [[屈原列传]] |
| 是以**见**放 | [[屈原列传]] |
| **见**犯乃死，重负国 | [[苏武传]] |
| 慈父**见**背 | [[陈情表]] |
| 遂**见**用于小邑 | [[归去来兮辞]] |
| 皆好辞而以赋**见**称 | [[屈原列传]] |

### 3. "见……于……" 式（"见"+"于"组合）

| 例句 | 出处 |
|------|------|
| 吾长**见**笑**于**大方之家 | —（《秋水》） |
| 遂**见**用**于**小邑 | [[归去来兮辞]] |

### 4. "为" 字式

| 例句 | 出处 |
|------|------|
| 身死人手，**为**天下笑者 | [[过秦论]] |
| **为**天下笑 | [[屈原列传]] / [[过秦论]] |

### 5. "为……所……" 式（最常见的有标志被动）

| 例句 | 出处 |
|------|------|
| 皆为陛下**所**成就 | [[苏武传]] |
| 数十年竟**为**秦**所**灭 | [[屈原列传]] |
| 不者，若属皆且**为所**虏 | [[鸿门宴]]（省略"之"） |

### 6. "被" 字式

| 例句 | 出处 |
|------|------|
| 信而见疑，忠而**被**谤 | [[屈原列传]] |
| **被**发行吟泽畔 | [[屈原列传]]（"被"通"披"，非被动） |

---

## 二、无标志被动句（意念被动）

形式上没有被动标志，但主语实际上是动作的承受者：

| 例句 | 说明 | 出处 |
|------|------|------|
| 虞常**生得** | 虞常被活捉 | [[苏武传]] |
| 一夫作难而七庙**隳** | 宗庙被毁灭 | [[过秦论]] |
| 兵挫地削 | 军队被挫败、土地被削减 | [[屈原列传]] |
| 得不焚 | 能够不被烧毁 | [[项脊轩志]] |
| 戍卒叫，函谷**举** | 函谷关被攻占 | [[阿房宫赋]] |

---

## 三、与"见"相关的特殊结构

"见"并非总是表被动——"见+动词"有时是偏指（指代第一人称）：

| 例句 | 释义 | 出处 |
|------|------|------|
| 慈父**见**背 | 父亲离开了我 | [[陈情表]] |
| 府吏**见**丁宁 | 府吏叮嘱我 | — |
| 君既若**见**录 | 您既然记着我 | — |

---

## 四、翻译方法

1. **有标志被动句**：保留被动结构
   - 为天下笑 → 被天下人耻笑
   
2. **无标志被动句**：翻译时加"被"
   - 虞常生得 → 虞常**被**活捉
   - 七庙隳 → 宗庙**被**毁灭
