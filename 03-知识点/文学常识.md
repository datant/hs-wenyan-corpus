# 文学常识

> 文言文相关的文学体裁、流派、作家作品等基础知识。

---

## 一、常见文体

| 文体 | 说明 | 代表作品 |
|------|------|----------|
| 论 | 议论说理 | 《过秦论》《六国论》 |
| 说 | 阐述事理 | 《师说》 |
| 赋 | 铺陈描写，介于诗与散文之间 | 《赤壁赋》《阿房宫赋》 |
| 记 | 记叙描写 | 《登泰山记》《石钟山记》 |
| 书 | 书信 | 《与妻书》《答司马谏议书》 |
| 序 | 赠序/书序 | 《兰亭集序》《五代史伶官传序》 |
| 表 | 臣子向君主陈述衷情的文书 | 《陈情表》 |
| 疏 | 臣子向君主分条陈述意见的文书 | 《谏太宗十思疏》 |
| 传 | 人物传记 | 《屈原列传》《苏武传》 |
| 辞 | 韵文，偏重抒情 | 《归去来兮辞》 |
| 铭 | 铭文，刻于器物 | （课外） |
| 志 | 记事文体 | 《项脊轩志》 |
| 寓言 | 借故事说理 | 《庖丁解牛》《五石之瓠》 |

## 二、重要流派

| 流派 | 代表人物 | 核心主张 |
|------|----------|----------|
| 儒家 | 孔子、孟子 | 仁、义、礼、智、信 |
| 道家 | 老子、庄子 | 无为、自然 |
| 墨家 | 墨子 | 兼爱、非攻 |
| 法家 | 韩非 | 以法治国 |

## 三、史书体例

| 体例 | 说明 | 代表作品 |
|------|------|----------|
| 编年体 | 按年月日编排 | 《左传》《资治通鉴》 |
| 纪传体 | 以人物传记为中心 | 《史记》《汉书》 |
| 国别体 | 分国记事 | 《战国策》《国语》 |
